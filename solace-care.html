Solace Care Redesign

Solace Care is a planning and coordination tool for people who have recently lost someone. This redesign focuses on clarity, emotional accessibility, and lower cognitive load through a simpler and more focused interface.

Design goals

This redesign focuses on reducing mental effort, embedding AI support directly into tasks, and making shared planning easier.

Original checklist layout with time-based task grouping and category filters.
Original Solace Care home page with multiple entry points for support and AI assistance.

Opportunities for improvement

  • The original interface presents too many competing elements at once, increasing cognitive load.
  • AI assistance is key to the platform, but it is separate from the primary task flow.
  • Collaboration features are very limited.
Redesigned checklist view with expanded sections for the first days and weeks after a loss.

Redesigned checklist structure

The redesigned checklist keeps the original timeline structure but adds a dedicated section for the first days after a loss. The home page was removed to simplify navigation and make the checklist the primary entry point.

Checklist item layout combining task guidance, shared notes, and in-context AI assistance.

Collaboration and AI integration

A shared notes panel was added to each checklist item to enable in-context collaboration. This allows co-planners to communicate directly within tasks instead of relying on external tools.

The AI assistant is now available as a persistent chat panel across all pages. This replaces the previous separate view and floating action button, enabling context-aware support inside the active workspace.

The AI assistant was extended to reference not only articles and checklist items, but also user-uploaded documents.

Example of the AI assistant responding directly inside an active checklist task.

Reducing cognitive load

The interface was simplified to focus on what users need most. Unnecessary elements were removed to reduce distraction and make the experience easier to navigate.

Library view organizing support content, recommended reading, and a new search feature.

Document management

A dedicated documents page was added to provide an overview of important files. Previously, users had to navigate to a different area of the site to access documents uploaded by co-planners or the deceased.
