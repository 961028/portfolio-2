Solace Care Redesign

Solace Care is a planning and coordination tool for people who have recently lost a close relative. This redesign focuses on reducing complexity, improving clarity, and supporting emotionally sensitive users through calm interaction patterns.

Design goals

The project is guided by three core principles: low cognitive load, context-aware AI integration, and simple collaboration.

Original Solace Care start page.

Problems with the existing product

The current interface is visually cluttered, resulting in a high cognitive load. Although the platform aims to support AI assistance and collaborative planning, both features are poorly integrated.

New checklist view with expanded sections and a clearer hierarchy of tasks.

Design approach

The redesigned checklist keeps the original timeline structure but introduces a new section dedicated to the very first days after a loss. By separating these immediate tasks from those that come in the following weeks, it helps users focus on what needs attention right now without being overwhelmed by everything that needs to be done in the following weeks. In addition, toggles for hiding checked items and tasks assigned to others have been added.

Redesigned checklist item page, combining guidance text, shared notes, and a context-aware AI assistant.

A shared notes panel has been added to every checklist item page to support collaboration directly within the planning process. Co-planners can leave a note where it is most relevant, rather than relying on external tools.

The AI assistant has been integrated as a persistent chat panel across all pages instead of being isolated on a separate view or hidden behind a floating action button. This allows the AI to respond based on the user's current context, directly in their active workspace.

AI-assisted workflow example, where the assistant responds directly to the active context.

The overall interface has been reduced to core functionality only. Visual elements and content have been deliberately simplified to minimise distraction and maintain a low cognitive load, which is especially important for the target group.

Redesigned Solace Care library view where articles live.

Finally, a documents page has been added to provide a clear overview of all important files in one place. Currently, the user has to navigate to a separate part of the platform to access documents, uploaded by other planners or by the deceased while they were still alive.

Redesigned documents page, showing an organised file listand an expanding search bar icon.
